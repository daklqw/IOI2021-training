## GH
……好难

首先枚举位都是能想出来的，至于怎么匹配呢？

我们为了匹配变量，要去掉随机因素。

例如协方差，如果有 `cov(x, y)` 且 $x$ 和 $y$ 无关，且 $x$ 完全随机，则 `cov(x, y) = 0`。

依此，我们把已经确定的位贡献去掉，把剩下的位按照填 $0$ 和 $1$ 分别计算期望。

由于 $A$ 幂次大了类似随机，所以我们式子十分好列。

然后就直接对比协方差即可。
